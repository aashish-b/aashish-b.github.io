I'm a software engineer based in Toronto, Canada. I have multiple years of experience working with JavaScript, Go, Ruby, AWS, Docker, SQL, and React.

My most recent project was Triage, an open-source consumer proxy that solves head-of-line blocking for Kafka consumers, using a dead letter store and parallelism.

Triage is an open-source consumer proxy for Kafka developers. It solves head-of-line blocking due to poison pills and non-uniform latency.

Triage avoids data loss by saving corrupt messages in a dead letter store. It also increases parallelism to enable more consumer instances to maintain throughput in case of a delay.

You can deploy Triage on AWS Fargate using our CLI!
